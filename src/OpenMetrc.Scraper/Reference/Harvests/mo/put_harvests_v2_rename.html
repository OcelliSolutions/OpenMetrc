Harvests

Server: https://api-mo.metrc.com/

PUT /harvests/v2/rename

Permissions Required

View Harvests
Manage Harvests

Parameters

licenseNumber
The license number of the facility for which to return the list of renamed harvests.

Example Request

PUT /harvests/v2/rename?licenseNumber=123-ABC[
