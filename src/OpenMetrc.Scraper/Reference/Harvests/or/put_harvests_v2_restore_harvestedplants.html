Harvests

Server: https://api-or.metrc.com/

PUT /harvests/v2/restore/harvestedplants

Permissions Required

View Harvests
Finish/Discontinue Harvests

Parameters

licenseNumber
The license number of the facility for which to return the list of harvests restored.
