Strains

Server: https://api-ak.metrc.com/

GET /strains/v1/active

Permissions Required

Manage Strains

Parameters

licenseNumber
The license number of the facility for which to return the list of active strains.

Example Request

GET /strains/v1/active?licenseNumber=123-ABC
