Strains

Server: https://api-ak.metrc.com/

GET /strains/v1/{id}

Permissions Required

Manage Strains

Parameters

licenseNumber
optional		If specified, the Strain will be validated against the specified License Number. If not specified, the Strain will be validated against all of the User's current Facilities. Please note that if the Strain is not valid for the specified License Number, a 401 Unauthorized status will be returned.

Example Request

GET /strains/v1/201?licenseNumber=123-ABC
