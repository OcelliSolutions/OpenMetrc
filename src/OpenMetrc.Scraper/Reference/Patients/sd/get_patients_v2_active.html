Patients

Server: https://api-sd.metrc.com/

GET /patients/v2/active

Permissions Required

Manage Patients

Parameters

licenseNumber
The license number of the facility for which to return the list of active patients.
pageNumber
optional
The number of the data page from which to return data.
pageSize
optional
The number of records to return per page. Pagination is currently disabled by default. You can enable pagination on this query by specifying a value that does not exceed 20.

Example Request

GET /patients/v2/active?licenseNumber=123-ABC&pageNumber=1&pageSize=10
